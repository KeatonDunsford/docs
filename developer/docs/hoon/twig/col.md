# Cell `:` ("col")

The `:` ("col") twigs, `:-` and friends, are simple and regular.
All `:` twigs expand to `:-` ("colhep"), which makes a pair 
(just like the Lisp [`cons`](https://en.wikipedia.org/wiki/Cons) operator).
