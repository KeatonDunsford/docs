# `|:  "barcol"`

`{$brcl p/seed q/seed}`: form a gate with burnt sample.

## Expands to

```
=>  ^~  p
  |.  q
```

## Discussion

Note that `p` is a seed, not a moss; `|:` doesn't bunt your sample as
[`|=`](../tis) does.

## Syntax

Regular: *2-fixed*.
