# `!! "zapzap"`

`[%zpzp ~]`: crash.

## Produces

Nothing.  Always crashes, with type `%void`.

## Syntax

`!!`

## Discussion

`%void` nests in every other span, so you can stub out anything with `!!`.

## Examples

```
~zod:dojo> !!
ford: build failed
```
